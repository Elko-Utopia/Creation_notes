⚙️ 自定义配置

// 自定义监控配置
LightboxMonitor.configure({
  logPerformance: true,    // 性能日志
  logScrollLock: false,    // 滚动锁定日志
  logImageLoading: true,   // 图片加载日志
  showTimestamps: true,    // 显示时间戳
  groupLogs: true          // 分组显示
});

🚀 快速启动命令

// 控制台中直接使用:
LightboxMonitor.enableBasic()      // 基础监控
LightboxMonitor.enableFull()       // 完整监控
LightboxMonitor.getStats()         // 查看统计
LightboxMonitor.getPerformanceReport() // 性能报告
LightboxMonitor.disable()          // 禁用监控
